Doom Virtual Health v3.0 1.21.6+

为 Minecraft 实体注入独立于原版血量的虚拟血量系统，自动映射到 Bossbar。

特性

  • 自定义血量上限（×1000 mHP 精度）
  • 高频 tick 伤害检测 + 余数精度补偿
  • 可配置伤害倍率（dvh.damage_mult）
  • 装备加成（attribute_modifiers 自动扫描）
  • UUID hex 击杀归属
  • auto_init（/summon {data:{dvh:{...}}}）
  • 百分比血量判定（predicate）
  • 无敌控制
  • Bossbar 自动同步（支持 persist）

快速开始

# auto_init
/summon minecraft:zombie ~ ~ ~ {data:{dvh:{max_health:"5000",health:"5000"}},NoAI:1b}

# 创建 Bossbar
/execute as @n[tag=virtual_health_entity] run \
  function doom.bossbar:api/create {with:{name:'"Boss"',color:red,style:progress,visible:"true",targets:"@a"}}

# 扣血
/execute as @n[tag=virtual_health_entity] run \
  function doom.virtual.health:api/add_health {points:-500}

API

函数参数说明
api/create{with:{max_health, health, on_death?}}注入虚拟血量
api/remove{with:{kill:1b/0b}}剥离虚拟血量
api/add_health{points:int}增减血量
api/set_health{health:int}强制设定血量
api/set_max_health{max_health:int}设定最大血量
api/add_max_health{points:int}增减最大血量
api/get_health{out, scale}导出血量
api/get_max_health{out, scale}导出最大血量
api/get_stats{damage_out, healing_out, scale}导出统计
api/get_percentage{out}导出百分比 [0,100]
api/set_invulnerable{state:1b/0b}设置无敌
api/set_damage_mult{mult:int}设置伤害倍率
api/apply_trigger—分配 vitality trigger
predicate/is_invulnerable—判定无敌
predicate/below_percentage—判定低于 @s dvh.pp%
predicate/above_percentage—判定高于 @s dvh.pp%
predicate/between_percentage—判定在 @s dvh.pp~@s dvh.pp_max% 内

安装

  1. 下载数据包 zip（从 Releases）
  2. 放入存档的 datapacks/ 文件夹
  3. /reload 或重进世界
  4. 看到 [DVH] Virtual Health System Loaded. 即成功
